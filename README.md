# wvue

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# 启动express服务
node index

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

这是一个前端vue
